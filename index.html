Machine Learning AI

In Skit Gubbe I've experimented with using Machine Learning to develop an AI opponent. I first made a simple opponent that only played the lowest card possible but later I started looking in to Q-Learning AI, which is a type of reinforcement learning where the AI learns by trial and error while playing itself. it plays the game over and over and gradually figures out which actions tend to lead to winning. It does that by saving states to a Q-Table which it then references while playing. If it has not seen that exact position/state before it will save it to the Q-Table and if it's actions was good or bad which it decides through a reward system
